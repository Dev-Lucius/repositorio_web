Sobre o Projeto

O que você encontrará aqui

Este portfólio funciona como um registro contínuo da evolução técnica ao longo da disciplina. Cada projeto aborda diferentes aspectos do desenvolvimento web, desde páginas estáticas até interatividade com JavaScript.

A estrutura segue boas práticas modernas: semântica HTML, separação de responsabilidades (HTML, CSS e JS) e consistência visual em todas as páginas.

Estrutura Semântica

Uso correto de elementos como <header>, <main> e <footer>.

Design Consistente

Identidade visual padronizada com cores, tipografia e espaçamento bem definidos.

JavaScript Puro

Funcionalidades interativas desenvolvidas sem uso de frameworks externos.

Responsividade

Layout adaptável utilizando Flexbox, Grid e boas práticas de design responsivo.
